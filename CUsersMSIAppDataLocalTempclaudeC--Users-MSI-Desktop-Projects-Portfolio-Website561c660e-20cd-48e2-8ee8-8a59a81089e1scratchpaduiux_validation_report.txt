2. DARK MODE SUPPORT
====================
PARTIAL PASS - Spinner color (--acc) has dark variant working
FAIL - Desktop background (--desk) does NOT change in dark mode
FAIL - Window background (--winBg) does NOT change in dark mode
FAIL - Text color (--txt) does NOT change in dark mode

Issue Identified:
The init code (lines 961-963) sets inline styles for --desk and --winBg:
  document.documentElement.style.setProperty('--acc', CFG.accent);
  document.documentElement.style.setProperty('--desk', CFG.desktopBg);
  document.documentElement.style.setProperty('--winBg', CFG.winBg);

The setTheme() function (lines 968-975) only updates --acc and --deskWave inline,
but NOT --desk and --winBg. Since inline styles have higher CSS specificity than
[data-theme="dark"] selectors, the CSS cascade values never apply.

The CSS rules at lines 118-126 define [data-theme="dark"] colors but they're
being overridden by the inline light-mode values set during init.

Verification:
- data-theme="dark" attribute sets correctly ✓
- accentDk (#d18615) applies correctly ✓
- --desk remains #e0f2fe (light value) instead of #0f172a ✗
- --winBg remains #ffffff instead of #121212 ✗
- --txt remains #e8eacc (dark) in light mode (wrong) ✗

CRITICAL BUG: Dark Mode CSS Variables Not Updating
---
File: C:\Users\MSI\Desktop\Projects\Portfolio Website\index.html
Lines: 968-975 (setTheme function)

The setTheme() function must also update --desk and --winBg inline styles
to properly override the init-time values when toggling dark mode.

Current (broken):
  function setTheme(dark) {
    isDark = dark;
    document.documentElement.setAttribute('data-theme', dark ? 'dark' : 'light');
    document.documentElement.style.setProperty('--acc', dark ? CFG.accentDk : CFG.accent);
    document.documentElement.style.setProperty('--deskWave', dark ? '#1e293b' : '#bae6fd');
    themeBtn.textContent = dark ? '☀️' : '🌙';
    localStorage.setItem('theme', dark ? 'dark' : 'light');
  }

Should be:
  function setTheme(dark) {
    isDark = dark;
    document.documentElement.setAttribute('data-theme', dark ? 'dark' : 'light');
    document.documentElement.style.setProperty('--acc', dark ? CFG.accentDk : CFG.accent);
    document.documentElement.style.setProperty('--desk', dark ? CFG.desktopBgDk : CFG.desktopBg);
    document.documentElement.style.setProperty('--winBg', dark ? CFG.winBgDk : CFG.winBg);
    document.documentElement.style.setProperty('--deskWave', dark ? '#1e293b' : '#bae6fd');
